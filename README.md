# Website für die Alte Schule Bissingen o. L. – Kreativhaus 

Dies ist die offizielle Website für das **alte Schulhaus in Bissingen o. L.**, das heute als **Kreativhaus** genutzt wird.
Die Website informiert über die Räumlichkeiten, Nutzungsmöglichkeiten und Mietoptionen für Einzelpersonen und Firmen.

---

## Features

- Vorstellung des historischen Schulgebäudes
- Informationen über Nutzungsmöglichkeiten für Kreative, Workshops oder Studios
- Übersicht über Mietoptionen (Tages-, Wochen- oder Monatsweise)
- Responsive Design für Desktop, Tablet und Mobile
- Schnelle, statische Website für maximale Performance

---

## Tech Stack

- **Frontend:** HTML & CSS
- **Interaktivität:** Wenig JavaScript (für minimale Funktionalitäten)
- **Deployment:** Statische Seite, leicht zu hosten auf GitHub Pages

---
